Política de Privacidad

Última actualización: Marzo 2026

1. Introducción

Bienvenido a Anotalo. Esta Política de Privacidad describe cómo recopilamos, usamos y protegemos tu información cuando utilizas nuestra aplicación de gestión de finanzas personales.

2. Información que Recopilamos

Dato Recolectado Compartido con terceros Uso
Email Sí No (solo Supabase como backend) Autenticación, creación de cuenta
Nombre de perfil Sí Sí (con contactos en gastos compartidos) Identificación entre usuarios
Transacciones financieras Sí No Funcionalidad principal de la app
Device ID Sí No Sincronización offline-first
Push token Sí No (solo FCM como servicio) Notificaciones push
Audio (micrófono) Sí, temporal Sí (Google Gemini API) Transcripción por voz de transacciones
Datos de suscripción Sí Sí (Mercado Pago) Gestión de pagos

3. Cómo Usamos Tu Información

Utilizamos tu información exclusivamente para:

  • Autenticación: Tu email se usa para crear y gestionar tu cuenta.
  • Funcionalidad de la app: Tus transacciones, cuentas y categorías se almacenan para brindarte el servicio.
  • Sincronización: El Device ID permite sincronizar datos entre tu dispositivo y la nube.
  • Notificaciones: El push token permite enviarte notificaciones relevantes (gastos compartidos, etc.).
  • Transcripción por voz: El audio se envía a Google Gemini API para convertir tu voz en datos de transacciones. El audio no se almacena.
  • Pagos: Los datos de suscripción se comparten con Mercado Pago para gestionar cobros recurrentes.

4. Servicios de Terceros

Anotalo utiliza los siguientes servicios de terceros para su funcionamiento:

4.1 Supabase

Backend, base de datos y autenticación. Los datos almacenados incluyen: email, perfil, transacciones, cuentas, categorías, contactos y gastos compartidos. Los servidores se encuentran en la nube.

4.2 Google Gemini API

Procesamiento de audio para transcripción de transacciones por voz. El audio se envía, se procesa y no se almacena por Anotalo. Google tiene sus propias políticas de retención de datos.

4.3 Mercado Pago

Procesador de suscripciones. Recibe el email del usuario y datos de pago para la gestión automática de cobros recurrentes.

4.4 Firebase Cloud Messaging (FCM)

Servicio de envío de notificaciones push. Recibe el token del dispositivo para entregar las notificaciones.

4.5 Expo Updates

Servicio de actualizaciones over-the-air de la aplicación. Recibe metadata del dispositivo para entregar actualizaciones.

5. Sincronización y Almacenamiento

Anotalo utiliza una arquitectura offline-first:

  • Todos los datos se guardan primero localmente en tu dispositivo (SQLite).
  • Cuando hay conexión a internet, los datos se sincronizan a la nube (Supabase) para respaldo y para habilitar funcionalidades como gastos compartidos.
  • La app funciona completamente offline. La sincronización ocurre automáticamente cuando se recupera la conexión.

6. Gastos Compartidos

Al utilizar la función de gastos compartidos:

  • Tu nombre de perfil y los detalles de las transacciones compartidas son visibles para los otros participantes del gasto.
  • Los participantes pueden ver el nombre del creador del gasto, el monto y la descripción.
  • Esta información se comparte únicamente con los usuarios involucrados en cada gasto compartido.

7. Retención y Eliminación de Datos

  • Los datos se retienen mientras tu cuenta esté activa.
  • Al eliminar tu cuenta desde la app, se eliminan todos los datos asociados de forma permanente: perfil, transacciones, cuentas, categorías, contactos, gastos compartidos, tokens, notificaciones y cualquier otro dato vinculado a tu cuenta.
  • Los datos locales (SQLite) se limpian del dispositivo al eliminar la cuenta.
  • Audio: No se retiene. Se procesa y se descarta inmediatamente después de la transcripción.

8. Seguridad de los Datos

Implementamos medidas de seguridad para proteger tu información:

  • La comunicación con nuestros servidores utiliza cifrado HTTPS.
  • Las sesiones de usuario están protegidas con tokens seguros almacenados de forma encriptada en el dispositivo.

9. Tus Derechos

Tienes derecho a:

  • Acceder a tus datos personales.
  • Eliminar tu cuenta y todos los datos asociados desde la app.
  • Exportar tus datos financieros.

Para ejercer estos derechos, puedes hacerlo directamente desde la aplicación o contactarnos.

10. Menores de Edad

Anotalo no está dirigida a menores de 13 años. Si sos menor de 13 años, no debés usar la app sin la supervisión de un adulto responsable. No recopilamos intencionalmente datos de menores de 13 años.

11. Transferencia Internacional de Datos

Tus datos pueden almacenarse y procesarse en servidores ubicados fuera de la República Argentina (a través de servicios como Supabase, Google y Firebase). Al usar la aplicación, consentís la transferencia de tus datos a estos servidores internacionales.

12. Cambios a Esta Política

Podemos actualizar esta Política de Privacidad ocasionalmente. Te notificaremos sobre cambios significativos a través de la aplicación o por correo electrónico.

13. Contacto
